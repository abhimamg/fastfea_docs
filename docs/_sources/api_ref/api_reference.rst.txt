Post Processing
=================

.. |odb_file_desc| replace:: Name/path of the abaqus odb file. Wrap the path name within ``r'C:\path\file.odb'``, if the file is not in the same folder.
.. |step_desc| replace:: Step of interest in the odb file. This should be the name of the step. For example: ``Step-18``, ``LOAD``.
.. |frame_desc| replace:: Frame index number. This number is between 0 and total number of frames. This is not the same as the frame increment number which can be higher is the selected output frequency is greater than 1.
.. |part_desc| replace:: Name of the model part. This is usually ``Part-1-1`` unless explicitly defined in the model. 
.. |elset_desc| replace:: Element Set name. This should exclude the part name. For example element set for ``PART-1-1.CP_B_ELS`` would be ``CP_B_ELS``.
.. |ndset_desc| replace:: Node Set name. This should exclude the part name. For example element set for ``PART-1-1.CP_B_NDS`` would be ``CP_B_NDS``.
