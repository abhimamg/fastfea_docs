Post Processing

odbreader Module

The functions presented in this module extracts the the data from odb file and returns a python dictionary.

fastfea.abaqus.odbreader.get_ESF(odb_file, step, frame, part, elset, **kwargs)

Returns the Effective Axial Force (ESF1)

Parameters

  • odb_file (str) – Name/path of the abaqus odb file. Wrap the path name within r'C:\path\file.odb', if the file is not in the same folder.

  • step (str) – Step of interest in the odb file. This should be the name of the step. For example: Step-18, LOAD.

  • frame (int) – Frame index number. This number is between 0 and total number of frames. This is not the same as the frame increment number which can be higher is the selected output frequency is greater than 1.

  • part (str) – Name of the model part. This is usually Part-1-1 unless explicitly defined in the model.

  • elset (str) – Element Set name. This should exclude the part name. For example element set for PART-1-1.CP_B_ELS would be CP_B_ELS.

Returns

1D list of Effective Axial Force sorted by elements.

Return type

list
